Backend-разработка

Инфраструктура,

01

Микросервисы

Разрабатывал и поддерживал сервисы на FastAPI. Совместно с тимлидом проектировал структуру запросов/ответов для новых модулей.

02

Миграции БД

Проектировал модели (SQLAlchemy), писал миграции (Alembic). Провёл сложную нетривиальную миграцию с изменением типов полей — без даунтайма и потери данных.

03

Тесты и CI

Покрыл ~50% критичного функционала юнит- и интеграционными тестами на Pytest. Настроил их запуск в CI-пайплайне.

04

Интеграции

Подключал платёжный шлюз и сторонние API с обеспечением отказоустойчивости. Актуализировал docker-compose и писал техдокументацию по эндпоинтам.

04

AI-инженерия

LLM
в production —
не в ноутбуке

Участвовал в разработке AI-модуля — от пилотного скрипта до полноценного внедрения с фидбеком от продакта.

Написал надёжную обёртку для OpenAI/Anthropic API: таймауты, fallback-сценарии, повторные попытки. Настроил логирование потребления токенов и задержек.

RAG

Помогал проектировать пайплайн, чтобы LLM опиралась на внутреннюю базу знаний компании.

pgvector

Хранение эмбеддингов и семантический поиск прямо в PostgreSQL — без внешних векторных БД.

Celery

Вынес долгие LLM-вызовы в фоновые задачи, реализовал асинхронный polling статуса для фронтенда.

Redis Cache

Кэшировал семантические ответы LLM — минус 30% платных запросов к AI-провайдеру в пилоте.

MCP

Исследовал и тестировал интеграцию для безопасного доступа LLM-агентов к корпоративной БД.
